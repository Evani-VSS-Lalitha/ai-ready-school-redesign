# AI Ready School – Homepage Redesign

## Problems Identified
While analyzing the existing website, I found a few key UX issues:

- The hero section does not clearly explain what the product does.
- The 5 products are shown separately without explaining how they are connected.
- The call-to-action (Book Demo) is not very prominent.
- There are no strong trust signals like testimonials or school logos.
- The overall layout lacks proper spacing and visual hierarchy, making it harder to read.

---

## What I Prioritized
Based on these issues, I focused on improving the most important sections:

- Created a clear and simple hero section with a strong value proposition.
- Simplified the product section by presenting it as one ecosystem.
- Highlighted the main CTA (Book Demo) for better visibility.
- Designed a clean and minimal navigation bar for easy access.

---

## Improvements Added
- Added anchor links for smooth navigation between sections.
- Included hover effects to make the UI feel more interactive.
- Made the navbar sticky for better usability.
- Added a small trust line in the hero section to improve credibility.

---

## What I Would Do Next
If I had more time, I would:

- Add testimonials and logos of schools for better trust.
- Improve animations and micro-interactions.
- Create separate pages for each product.
- Further enhance the mobile user experience.
